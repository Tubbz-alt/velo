Which data should I share and preserve?

It is the Principal Investigator's (PI) responsibility to determine which data should be the subject of the Data Management Plan (DMP), and to propose which data should be shared and preserved. The following has to be considered:

  • what types of data the proposed research will generate
  • which data could have value to others
  • when will the data be shared
  • are any limits to data sharing required (e.g., intellectual property protection)
  • what data formats and quality standards will be applied to enable the data to be shared effectively.

Data Sharing

The Office of Science provides the following definition for Data Sharing :

"Data Sharing means making data available to people other than those who have generated them. Examples of data sharing range from bilateral communications with colleagues, to providing free, unrestricted access to the public through, for example, a web-based platform."

Data Preservation

Data preservation means providing for the usability of data beyond the lifetime of the research activity that generated them.
