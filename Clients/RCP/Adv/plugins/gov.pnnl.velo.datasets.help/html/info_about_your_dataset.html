Information About Your Dataset

Information about your dataset is needed for purposes of its retrieval in dataset search engines like Data.gov and for populating the PNNL landing page. Velo was designed with the U.S. Government's Project Open Data in mind, and the fields below reflect those in the associated schema.

There are required and optional fields. Clicking on a hyperlink will take you to the page for the metadata field where you can read a description about it and view examples.

Required Fields

  • Access Level
  • Listing Date
  • Authors
  • Program Code
  • Bureau Code
  • Publisher
  • Contact Name
  • Tags (Keywords)
  • Contact Role
  • Title
  • Description
  • Unique Identifier
  • Last Updated
  

Optional Fields

  • Categories
  • ORCiD
  • Data Quality Summary
  • Related Websites
  • License
  • Spatial/Geographical Coverage
  • Metadata Standards
  • Temporal Coverage
  • Publications
  • Update Frequency
