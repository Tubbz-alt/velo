Program Code

Represents the Federal program responsible for the dataset. The source of the program code is the Federal Program Inventory Reference Table. More than one program code is allowed.

A link to the most current Table is available on the home page for the Federal Program Inventory under the section Additional Resources. The home page will not open in this window. Please copy and paste the following URL into your browser's address bar.

http://www.performance.gov/federalprograminventory

Example

The Department of Energy's Office of Biological and Environmental Research would be entered as 019:025.
