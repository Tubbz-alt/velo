Listing Date

The date when the resource was or will be made publicly available. In the case of datasets, “publish” is understood to mean making the data available on a specific date to the community of researchers.

Format: YYYY-MM-DD. (ISO 8601 standard) The time does not need to be included.

If the date cannot be determined, use the date when a digital object identifier (DOI) was requested.

If an embargo period has been in effect, use the date when the embargo period ends.

If there is no standard publication year value, use the date that would be preferred from a citation perspective. This would be the case for data that is streaming.
