Access Level

The degree to which this dataset could be made publicly-available, regardless of whether it has been made available.

Choices:

  • Public (The dataset is or could be made publicly available without restrictions).
  • Restricted public (The dataset is available under certain use restrictions).
    • Examples of user restrictions:
      • Attribution requirement.
      • Payment requirement.
  • Non-Public (The metadata about the dataset is available to the public, however the actual dataset is not.)
    • Examples:
      • Twitter-feed data.
      • Data with personally identifiable information that cannot be managed.
