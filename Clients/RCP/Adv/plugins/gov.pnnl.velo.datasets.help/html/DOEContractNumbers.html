DOE Contract Number(s)

Enter the DOE contract number under which the work was funded. If the dataset is a result of a joint effort between two or more DOE Site/Facility Management Contractors, etc., additional DOE contract numbers may be entered. The "DE" should not be included as a part of the number. When more than one number is entered, the first number is considered the primary number. If a DOE Contract Number does not apply for the originating organization, the word "NONE" should be entered.

Example

If the contract number is: DE-AC12-34RL56789, you would enter: AC12-34RL56789.

Entries with multiple contacts would look like:

AC12-34RL56789; AC23-45RL67891; ...
