Landing Page Configuration

There are required and optional fields to complete the configuration of the landing page for your dataset.

<images(s) of dataset editor. Depending on layout, maybe the elements should be grouped with the fields?>

Required Fields

  • Access Level
  • Listing Date
  • Authors
  • Program Code
  • Bureau Code
  • Publisher
  • Contact Name and Email
  • Tags (Keywords)
  • Description
  • Title
  • Last Updated
  • Unique Identifier

Optional Fields

  • License
  • Data Quality Summary
  • Publications
  • Metadata Standards
  • Spatial/Geographical Coverage
  • Related Websites
  • Temporal Coverage
  • Update Frequency
  • Categories
