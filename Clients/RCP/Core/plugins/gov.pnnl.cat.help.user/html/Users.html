Users Perspective

Users Perspective Structure

The Users perspective provides a list of of Velo users and additional identifying information that each user can add about themselves, such as name, email, phone, and a picture.

The Users Perspective is organized in two different views (Figure 1):

  • Users View
  • Users Details

Figure 1. Users Perspective Structure.

 

Users View

The User View provides the list of Velo's users.

Only system administrators are allowed to add new users by clicking on . By default, the user account name appears in this list unless a different name is specified. Initial letters of a user name in the “filter text” box can be typed to see a reduced list of users based on the filter criteria (Figure 1).

Double click on the user name to open a dialog for modifying user properties. In the Person window, user information can be modified (e.g., email, picture) (Figure 2, [1]). The Account password can be modifed in the Account window (Figure 2, [2]). The Account window provides also the list of teams the user belongs to.

 

Users Details

The Users Details View provides identifying information such as name, email, phone, and a picture (Figure 1).
