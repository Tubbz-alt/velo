Home Perspective Structure

The Home perspective is the first page the user sees when Velo is launched (Figure 1). Its purpose is to provide a gateway to the Velo platform. From this window basic features of the application can be accessed, which include

  • Getting Started which, whick links to the general Help of Velo Data Management Platform.
  • Browsing Data Repository, which links to the Data Browser perspective
  • Searching Repository, which links to the Search perspective
  • Creating a Team, which links to the Teams perspective

Clicking on theÂ Report a Problem link, however, does not result in opening a new perspective. Rather, an email link is provided for reporting issues with the software (email to support.velo@pnnl.gov).

Users interested in learning more about Velo's capabilities is invited to visit the Velo's page by clicking on Get More Information on Velo .
