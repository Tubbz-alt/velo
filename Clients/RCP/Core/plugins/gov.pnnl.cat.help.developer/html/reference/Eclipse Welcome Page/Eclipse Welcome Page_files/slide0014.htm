Differences between static intro and dynamic intro.
lDynamic intros have anchors where additional content can be add by other plug-ins.  Static intros don’t allow any external contributions to be added during programs execution.
lThe dynamic intro project has the anchor tag in the root.xhtml file, the org.eclipse.ui.intro.config has already been extended by an org.eclipse.ui.intro.configExtension, and there is a new file called ext.xml which has been configure in the plug-in.xml file to the new anchor point.
