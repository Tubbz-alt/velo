Currently in Eclipse
lThe Eclipse application defines the splash screen in it’s config.ini file.
l
lThe splash file must be of type .bmp and have the name “splash” (i.e. splash.bmp).  This is not configurable since it is defined in the Product Branding plug-in extension itself.
l
lThe config.ini file allows the user two different ways of specifying the splash files location.
Either as a common-separated list of locations to search;
l                 osgi.splashPath = platform:/base/plugins/org.eclipse.platform
Or, as a direct location;
l         osgi.splashLocation = platform:/
l
lNote: The current location for the splash file is within the folder “eclipse\plugins\org.eclipse.platform_3.1.2” therefore I am assuming from the above search path that anything past the underscore (the version number) is ignored.
l
lNote: If both configurations are defined the osgi.splashLocation takes precedent.
l
lNote: When the search is used the finally resolved location is mapped to the osgi.splashLocation Key.
